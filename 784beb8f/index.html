Mac iTerm2登陆CentOS提示warning: setlocale: LC_CTYPE: cannot change locale (UTF-8): No such file or directory

2019-07-02 Tools iTerm2 评论

【报错原因】：没有utf-8这个语系（没添加语言_国名前缀），LC_ALL又没设定值。 服务端解决方法： 在远程系统上， /etc/environment加入以下两行，重新登陆即可。

LANG=en_US.utf-8
LC_ALL=en_US.utf-8

Mac终端解决方法： 编辑~/.bashrc或者~/.zshrc文件，添加

export LC_ALL=en_US.UTF-8  
export LANG=en_US.UTF-8
